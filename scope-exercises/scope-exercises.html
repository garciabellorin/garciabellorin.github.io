2. What is the difference between a method and function?

The main difference is that functions are a set of methods that perform a task whereas a method is a function assigned to an object property. Functions can be used directly while methods can be called only by the object that contains them.

For example, if we have a function to calculate area

var rectangle = function area (sideA, sideB) {
    var sum = sideA * sideB;
    return sum;
};

We can use area function as a standalone object because it is not associated with any other object, just the required parameters. For example, rectangle(2, 4); // 8

However, if we have a simple object like a string (var str = 'TWO WORDS') and we want to use the method toLowerCase(), we have to associate it with the given string, otherwise it does not have any sense var lowCase = str.toLowerCase(); // 'two words' since toLowerCase()is a method of String object.

3. What does 'this' refer to when used in a Java method?

In Java, this refers to the instance variables of a class, disregarding the method from what this variable is called.

4. What does 'this' refer to when used in a JavaScript method?

In JavaScript, this refers to the variables of the object owner when a method is called. However, the method itself can have its own variables and, at the same time, every inner object can have its own variables to which they also refer with a this keyword.

5. What does 'this' refer to when used in a JavaScript constructor function?

Refers to the parameters of a function constructor. Initially, this does not have a value but it is used to pass the parameters references to the object variables, similar to Java. Later, the object itself uses them.

7. What is a free variable in JavaScript?

Free variables are variables that are neither locally declared nor passed as parameter. They are outside of the closure. However, a free variable for one method might not be a free variable for an outer method.

10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments. 

function aFunction (...numbers) {
    var product;
    for (let i = 0; i < numbers.length; i++) {
        product *= numbers[i];
    }
    return product;
}
