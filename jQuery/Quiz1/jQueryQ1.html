jQuery Quiz I

  1. In jQuery terminology, what is the difference between the "jQuery function" and the "jQuery object"?

    Functions are standalone entities that require an input, not necessarily declared, to apply a method or a set of methods on it, and it is also not necessary to be related to the DOM. For example: $.trim(element);

    Objects are required to be grabbed by a selector and the returned value must be chained to another method. For example, $("p").hide();

    The main difference is that functions apply methods to an input whereas objects are required to be grabbed and chained to methods to obtain a value.
