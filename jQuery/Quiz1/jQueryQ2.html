  1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 
    2. //code fragment 1
$("li").each(function(idx, e) {
    $(e).css(“color”, “yellow”); });

//code fragment 2
$("li").each(function() {
    $(this).css(“color”, “yellow”); });

//code fragment 3
$("li").each(function(idx) {
    $(this).css(“color”, “yellow”); });

    Answer:

    All three are equivalent. Working with the .each() function give us some flexibility. This function can work seamless with objects and arrays and uses (index, element) as parameters but in our case we are not interested on (not using) the index value since we iterate trhough all the elements.

    This is why on code fragment 1 we use the element paramenter to grab the object and on the other two code fragments, disregarding of the parameters we use this to grab the object itself, this represents the owner of the method, in this case, all listed (li) items, and still get the same result.
